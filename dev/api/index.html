Notes:

  • Moving models are quasi-steady in this release.
  • storage != 0 is rejected for moving geometry.
  • Moving two-phase models require flux_jump = 0.

Internal API

PenguinDarcy._apply_box_bc_darcy! — Function

Apply outer box BCs for Darcy pressure equation with flux semantics:

  • Dirichlet: p = p_D
  • Neumann: u⋅n = q_n
  • Robin: αp + β(u⋅n) = g

where u = -Λ(∇p - ρg). In v0.2 we use the normal mobility component for the boundary elimination formula.
